In JavaScript .sort() orders the array alphabetically.

The solution to achieve a numeric sort is passing a comparison function to the .sort() method as a parameter.

Also, the .sort() method sorts the array in-place (mutates the original array),

so we should make a copy while sorting by using the syntax [...array].sort().

If speed is really important, we can use a TypedArray.
